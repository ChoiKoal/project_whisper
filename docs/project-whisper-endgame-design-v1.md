## 0. 한 줄 목표

> **v0.9.0에서 다섯 세계를 정화하고 다섯 포탈을 전점등(`five_portals_lit`)시켰다. 이제 게임의 '끝'을 설계한다 — 홈 섬 중앙에 여섯 번째 「빛의 문」을 세우고, 그 문 앞에서의 단 하나의 선택으로 두 엔딩을 가른다.**

핵심 축:
- **들어간다 → E1「완성」** — 임무를 완수한다. 장엄하지만, 마지막 컷에서 새소리가 두 번 반복된다(정체의 첫 징후). 선배들의 실수를 반복했다는 무언의 암시.
- **돌아선다 → E2「속삭임」(트루)** — 임무를 거부하고 세계에 남는다. 단, **진상 조각 5개**를 모아 진실을 알고 있어야만 '돌아서다'가 열린다. 크레딧 후 세계가 대답한다.

두 엔딩의 대비 = 스토리라인 §2 진상의 시스템적 번역:
> *"선배들은 세계를 완성하고 떠났기 때문에 세계가 시들었다. 완성 = 속삭임의 중단."*
E1은 그 실수의 반복(완성=정체), E2는 그 고리의 거부(미완=계속되는 속삭임).

## 1. 빛의 문 (Door of Light)

### 1.1 트리거 — `five_portals_lit` → 여섯 번째 문

- 접속점: `game/scripts/core/game_state.gd`
  - `signal five_portals_lit()` (구현됨) — 다섯 레이어 전 정화 시 1회 발화, `light_gate_previewed_flag`로 세이브 라운드트립.
  - `maybe_light_five_portals()` — 멱등, `five_layers_purified()`(L1 `SaveManager.cleared` + L2~L5 `layerN_purified_flag` 5-AND) 충족 시 `set_all_portals(PORTAL_OPEN)` + 시그널 발화.
- **v0.9.0 현황**: 이 시그널은 이미 발화하지만, "빛의 문 **예고**"(카타드랄 컷신 텍스트 카드)까지만 구현돼 있고 **홈 섬에 실제 문을 스폰하는 코드는 없다.** ← EG-1의 핵심 작업.

**설계**: 홈 섬 세션(`game/scripts/world/home_session.gd`)이 `five_portals_lit`(또는 부팅 시 `light_gate_previewed_flag == true`)을 관측하면, **홈 섬 중앙 = 다섯 포탈 반원의 초점**에 여섯 번째 문 `LightGate`를 스폰한다.

```
        [자연]   [과학]   [기계]   [마법]   [신성]      ← 기존 5포탈, 반원 배치 (전부 OPEN)
              \      \      |      /      /
                    ● 빛의 문 (반원의 초점, 홈 섬 중앙)
                         ▲
                     플레이어 각성 대(臺) 근처
```

- 반원의 초점 = 다섯 포탈이 바라보는 기하학적 중심. 좌표는 홈 섬 세션의 dais(각성 대) 배치와 정합시켜 확정(EG-1 구현 시 실측). 다섯 포탈의 sigil이 전부 이 문을 향하도록(선택적 연출 — 저비용이면 채택).

### 1.2 비주얼 — 기존 포탈 리컬러 + 백금 발광

빛의 문은 **신규 아트가 아니라 `Portal`(`game/scripts/world/portal.gd`) 지오메트리·파티클 재사용 + 컬러 오버라이드**:

| 부품 | 기존(보라) | 빛의 문(백금) |
|---|---|---|
| `_veil`(소용돌이 베일) | `VIOLET` 계열 | 백금 `Color(1.0, 0.96, 0.82)` — 흰-금 |
| `_rune_glow` / `_sigil_glow` | 보라 발광 | 백금 톤 발광(가산블렌드 유지) |
| `_pool`(바닥 발광 풀) | `light_pool_violet.png` 보라 | 백금 틴트(같은 텍스처 self_modulate) |
| sigil 문양 | 레이어 모티프(잎/별/톱니/룬/후광) | **다섯 문양의 합** 또는 무문양 백금 원판(광휘) |
| 스케일 | 1.0× | **~1.4~1.5×** — 최종 관문 위압감 |
| 애니메이션 | sigil bob | 문 전체 느린 부양 + 저주파 맥동 |

- 재사용 부품: `Portal`의 상태별 렌더(`_render_open` 계열), `GlowSprite`(가산블렌드 z=5, `glow_layer` 그룹), `LightPool`. 신규 자산 = 백금 `light_pool_platinum.png` 1장(기존 violet 텍스처 리컬러) 정도.
- **주의(연출 정본 CS-06)**: 빛의 문은 "장엄"해야 하지만, E1의 함정(완성=정체)을 시각적으로 미리 흘리지 않는다. 문 자체는 완벽하게 아름답다 — 반복의 징후는 **엔딩 안**에서만 드러낸다.

### 1.3 진입 프롬프트 — 확인

플레이어가 빛의 문 진입 존(`Portal.ENTRY_*` 패턴 재사용)에 들어서면, 즉시 씬 전환하지 않고 **확인 프롬프트**:

> **「들어가면, 세계가 완성된다.**
> **…정말?」**
> **[ 들어간다 ]   [ 돌아선다 ]**

- 이 단일 프롬프트가 **E1 vs E2의 유일한 분기점**이다.
- `[돌아선다]` 선택지의 **가용 여부는 진상 조각 5개 보유에 종속**(§3.1). 조각 미완이면 `[돌아선다]`는 회색(잠김) + 툴팁: *"아직… 이 세계를 두고 떠날 이유를 다 알지 못한다."* → 진상 조각이 E2의 게이트임을 무언으로 안내(트루 엔딩 티저).
- `[들어간다]`는 **항상 가용**(E1은 무조건 진입 가능 — Balance 판정 §4 참조).
- 프롬프트 UI는 기존 모달/텍스트 카드 시스템(`opening.gd`의 Label+VBox 패턴) 재사용. `GameState.set_control_lock(true)`로 입력 잠금.

## 2. E1「완성」

**조건**: 빛의 문 진입 → `[들어간다]`. (Balance 무관, 항상 가용 — §4)

**연출 시퀀스** (CS-06 E1 정본 + 스토리라인 §5 E1):

1. **진입 플래시** — 백금 화이트 플래시(1 프레임 블룸, `clear_sequence.gd`의 화이트 플래시 재사용) → 화면이 백금빛으로 가득 참.
2. **완성 몽타주** — 다섯 세계의 **정화 순간 플래시백 카드** 5장이 순차 크로스페이드:
   - 카드1 (자연) — 어린 세계수를 되심는 순간
   - 카드2 (과학) — 관제탑에 불이 들어오는 순간
   - 카드3 (기계) — 대시계가 다시 도는 순간
   - 카드4 (마법) — 최심부 봉인이 다시 짜이는 순간
   - 카드5 (신성) — 대제단이 처음 '대답'을 받는 순간
   - 각 카드 = 텍스트 카드 시스템(`opening.gd` 타이밍 규격: FADE_IN 1.0 / HOLD 2.5~2.6 / FADE_OUT 0.9, 백금 틴트 드리프트) 재사용. 이미지는 저비용이면 레이어 무드 컬러 + 한 줄 카피, 여유되면 각 레이어 대표 스프라이트 1컷.
   - **대사 최소** — 몽타주는 "완성했다"는 성취감을 소리·빛으로만.
3. **마지막 컷 (정본 준수 — CS-06 그대로)**:
   - 내 섬으로 화면 복귀. 아주 천천히 정지 프레임으로.
   - **내 섬의 새소리가 두 번 반복된다.** 무음 배경 위, 대사 없음.
   - 두 번째 반복에서 화면이 미세하게 채도 저하(1세계 "시간 정지"의 첫 징후 — §4 Nature 독주 멸망 연출의 축소판, 하지만 여기선 Balance 무관하게 **완성 그 자체가 정체의 씨앗**임을 암시).
   - > *플레이어에게 말로 설명하지 않는다. 반복이 시작됐다는 걸 소리로만.* (CS-06 원문)
4. **크레딧**.
5. **NG+ 제안** (§5).

**서사 의미**: E1 = 1회차의 자연스러운 엔딩. 플레이어는 임무("제0세계를 완성하라")를 완수했고 — 정확히 그래서 선배들의 실수를 반복했다. 새소리 2회 반복이 그 무언의 고발. 이것이 회차(NG+) 동기이자 E2로 가는 서사적 갈고리.

## 3. E2「속삭임」(트루 엔딩)

**조건** (스토리라인 §5 트루 조건을 v1.0 스코프로 확정):
> **진상 조각 5개 전부 회수 + 빛의 문 앞 프롬프트에서 `[돌아선다]` 선택.**

- 스토리라인 §5 원문은 "진상 조각 5개 + Balance '불완전' 유지 + 완성 선언 거부". **v1.0에서는 Balance 조건을 드롭**(근거 §4: v1.0 Balance는 판정 게이트가 아니라 다양성 지표로 단순화). 즉 **진상 조각 5개 + 돌아선다** = 트루. 이 단순화가 트루 엔딩을 "진실을 다 알고, 그럼에도 남는다"라는 순수한 서사 선택으로 만든다.
- 진상 조각 미완이면 `[돌아선다]`가 잠겨 있으므로(§1.3), E2는 **구조적으로 조각 수집을 강제** → 스토리라인 §5의 "E2는 진상 조각이 필요해 2회차 이상 유도" 정합.

### 3.1 진상 조각 (Truth Shards) = 각 레이어의 서사 오브젝트

스토리라인 §2 진상은 "각 레이어의 유산 조각으로 분할 회수"된다. 이를 **기존 조사형 오브젝트 5종에 '조각 플래그'를 부여**하는 것으로 구현(신규 오브젝트 최소):

| # | 레이어 | 오브젝트 | 코드 위치(실측) | 기존 로그 데이터 |
|---|---|---|---|---|
| 1 | L1 자연 | **세계수 잎** | `game/scripts/world/world_tree.gd` (`object_id="world_tree"`) | 세계수 조우 텍스트 |
| 2 | L2 과학 | **마지막 로그 스크린** | terminal_station 데코(전용 스크립트 없음 — 스캐터) | 신규 로그 텍스트 필요 |
| 3 | L3 기계 | **멈춘 로봇** | `clockwork_city.gd` `ROBOT_LOGS`(5줄) | 있음 |
| 4 | L4 마법 | **마법사 잔영** | `mage_tower.gd` `MAGE_LOGS`(5줄) | 있음 |
| 5 | L5 신성 | **석화된 피조물/순례자** | `cathedral.gd` `STATUE_LOGS`(5줄) | 있음 |

- **관찰**: L3/L4/L5는 이미 `*_LOGS` 배열(각 5줄)로 진상 대사를 갖고 있다. L1(세계수)·L2(로그 스크린)만 조각 대사 신규 필요.
- **인터랙션**: 기존 조사형(InteractionController가 `object_id` 감지 → E키 조사 → 로그 표시) 재사용. **추가할 것 = 조사 완료 시 조각 플래그 세우기 + 도감 기록 탭에 항목 남기기.**

### 3.2 조각 플래그 & 도감 기록 탭

- **플래그**: `GameState`(또는 신설 `LoreManager`)에 `truth_shards: Dictionary`(예: `{"world_tree": true, "l2_log": true, ...}`) + `truth_shard_count() -> int`. 세이브 v2에 `"truth_shards"` 키 편입.
  - 조각 5개 = `truth_shard_count() == 5`. 이 값이 `[돌아선다]` 가용 판정 + E2 게이트.
- **도감 기록 탭**: 현재 Codex는 items/recipes만 기록(기록 탭 없음 — Explore 실측). **신규 = 도감에 「기록(진상)」 탭** 1개 추가:
  - 각 조각 회수 시 항목 + 전문(全文) 로그 열람.
  - 5개 완성 시 **최종 회수 카드**(스토리라인 §2): *"선배들 역시 '다음 세계'라는 임무를 받았을 뿐. 악의는 없었다. 시스템이 그렇게 설계돼 있었다. 그리고 너에게도 같은 임무가 내려와 있다 — '제0세계를 완성하라'."* → 이 카드가 뜬 뒤부터 빛의 문 `[돌아선다]`가 해금.
  - 재열람: 도감→기록 탭(정식, CS 스킵/재감상 규격과 동일).

### 3.3 E2 연출 시퀀스 (CS-06 E2 정본 준수)

빛의 문 앞 `[돌아선다]` 선택 →

1. **문이 조용히 닫힌다** — 빛의 문 `_veil` 알파/스케일 역재생(OPEN→DORMANT 전이 재사용), 백금 광휘가 잦아듦. 무음에 가까운 저음.
2. **섬의 모든 배치물이 한 번씩 반짝인다** — 홈 섬의 플레이어 배치 오브젝트 전체를 순차/일제 글로우 펄스(`GlowSprite`·`LightPool` 맥동 재사용). "세계가 응답을 준비하는" 신호.
3. **어린 세계수가 잎을 기울인다** — 프롤로그의 그 나무(1레이어에서 심은)가 어느새 내 섬에도 자라 있다. `world_tree` 스프라이트(또는 홈 섬용 축소 인스턴스)의 잎이 한 번 기욺(작은 회전 Tween).
4. **대사 (정본 그대로)**:
   > **「완성하지 않을 거야. 계속 말을 걸 거야.」**
5. **크레딧**.
6. **크레딧 후 — 오프닝 회수**: 검은 화면에 **보라 점 하나가 두 번 깜빡인다.** 오프닝(CS-01)과 같은 리듬. **이번엔 대답이다.** (오프닝의 "부름"에 대한 세계의 "응답" — Whisper 주제의 완결: 정화=신에게 처음 대답해준 것 → 이제 세계가 나에게 대답한다.)
   - 재사용: `opening.gd`의 보라 점 깜빡임 애셋/타이밍 그대로.
7. **NG+ 제안** (§5).

**서사 의미**: E2 = 진짜 엔딩. 진상을 다 알고(조각 5), 임무의 정체를 알고, **그럼에도** 완성을 거부하고 남는다. 미완의, 그러나 계속 자라는 세계. 크레딧 후의 대답 = 게임 전체의 논지("말을 거는 한 세계는 시들지 않는다")의 증명.
