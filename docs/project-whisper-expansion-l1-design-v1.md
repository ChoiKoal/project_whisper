# Part A — 구역 2 「고요의 화원」 40×40 타일 설계

## A-1. 컨셉

시작의 숲에서 세계수를 정화한 방랑자가, 숲 북쪽 오솔길을 따라 다다르는 곳. **반쯤 무너진 꽃의 신전과 그 안뜰**이다. 한때 온갖 색으로 피어 있던 정원은 색을 잃었다 — 꽃은 형태만 남고 잿빛으로 바랬으며, 벽화는 가루가 되어 색 모래로 흩어졌고, 물감을 개던 정원사는 손을 멈춘 채 석상이 되었다. 이 세계에 **색을 돌려주는 것**이 화원의 정화다.

시작의 숲이 "생명(자연)을 되살리는 곳"이었다면, 화원은 그 자연에 **색을 입히는 곳** — 물감·꽃즙 계열이 주역이 되는, L1 안의 작은 색채 챕터다. level-design B-3의 "붉은 꽃 신전·나비 추적·물 반사 눈속임" 취지를 계승하되, MVP 부담이 큰 나비 추적·왜곡 연출은 빼고 **색 조합(물감) 퍼즐**로 구체화한다(정적 스프라이트 눈속임 전제, GDD §3.2 경고 준수).

무드: 시작의 숲 초록 base 위에 **바랜 파스텔**(잿빛 섞인 분홍·연보라). 정화가 진행될수록(게이트를 풀수록) 채도가 돌아온다 — GA1 앞은 거의 무채색, 신전(최북)은 무지개. **엔진 틴트(CanvasModulate) + 리컬러**로 팔레트를 늘리지 않고 표현(아트가이드 §2·§3). 시작의 숲과 뚜렷이 구분되는 **"색이 빠졌다가 돌아오는" 구역**.

**진입점**: 시작의 숲 세계수 구역(북쪽)에서 오솔길로 연결 → 화원 남쪽 스폰 (19,39). 시작의 숲 클리어(G4) 후 개방(북쪽 오솔길에 색이 어렴풋이 비치기 시작 = 개방 신호). 왕래는 홈/구역 포탈 패턴 재사용(§Part C).

# Part A(2) — 구역 3 「생명의 심장」 40×40 타일 설계

## A-4. 컨셉

시작의 숲에서 세계수를 정화하면, 세계수 **뿌리 아래로 내려가는 길**이 열린다 — 세계 나무의 심부, **생명의 심장**이다. L1의 **후반 구역**(정화 후 개방)이자, 시작의 숲이 시작한 "생명 되살리기"의 **근원에 닿는 곳**. level-design B-3의 "생명의 심장(허브)" 취지를 계승하되, 이 확장에서는 4방향 허브 분기 대신 **세계수 심부로 하강하는 수직 구역**으로 구체화(허브 분기는 L2~L5 실제 레이어가 대체하므로, L1 확장 내에서는 심부 서사·생명 재획득·진상 조각에 집중).

세계수 뿌리 지형이 맵을 감싼다 — 뒤엉킨 뿌리, 뿌리를 타고 흐르는 생명수의 잔재(도랑), 가장 깊은 곳의 심장 코어 방. **고도 -1(지하 느낌)은 실제 고도가 아니라 틴트로** 표현한다(§A-6 아래 주석): 심부로 갈수록 화면이 어둑해지고 뿌리 사이 보라 발광이 강해지는 CanvasModulate 커브. L4/L5의 실제 고도 시스템은 쓰지 않는다(L1 확장은 평면 + 틴트로 충분, 로더 무수정).

여기서 방랑자는 두 가지를 얻는다: (1) **생명(Vita) Whisper 재획득처**(생명의 샘물 E) — 엔딩 Balance 4축(자연/에너지/마력/생명) 중 생명을 L1 내부에서 재확보(§A-6.2). (2) **진상 조각 확장** — 「선배 컨스트럭터의 첫 실험 흔적」과, 세계수에 얽힌 **첫 컨스트럭터의 잔향**(잔재 NPC). 시작의 숲 세계수(EG 진상)의 이야기를 **심부에서 한 겹 더** 연다.

무드: 시작의 숲 초록에 **심부의 어둠 + 뿌리 보라 발광**. 신비의 물(m, source10 T5M glow violet)을 뿌리 도랑에 재활용. 최심부는 심장 코어의 따뜻한 발광. **진입점**: 시작의 숲 세계수(북)에서 뿌리로 하강 → 심장 남쪽 스폰 (19,39). 시작의 숲 클리어가 개방 조건(세계수 정화 = 뿌리 길 열림).

## A-6.1 게이트 순서 강제 (BFS)

스폰에서 4방향 BFS. 벽 = void(V) + 물(~) + '현재 닫힌' 게이트 병목 셀. 게이트를 공간 순서대로 열며 도달 구역 확장. 채집/기능 오브젝트 셀은 인접 채집이므로 통과 허용(게이트 아님).

**고요의 화원** (probe = 각 게이트 북쪽 지대 진입점):

| 초기(아무것도 안 함) | O | **X** | X | X | X |

## A-6.2 재료 도달성 — 각 게이트 열쇠 재료 소스가 게이트 前 구역 (softlock 불가)

**불변식**: *N번째 게이트의 재료는 N-1번째 게이트까지 열린 영역에서 전량 확보 가능.* 아래 표의 "게이트 前?" 열이 모두 ✅ → 뒤 지대 의존 없음 → softlock 불가(`l1x_recipes.py` softlock 블록 = 전 게이트 `OK`, `RESULT: PASS`).

## A-6.3 생명(Vita) Whisper 재획득처 — 엔딩 3속성 대비 (order-safe)

endgame Balance는 4축(자연/에너지/마력/**생명**)이며, E1/E2 판정에 생명 보유가 관여한다. 시작의 숲 G4가 생명을 처음 주지만(economy: 생명=L1 자연 계열), 회차·진행에서 **소진**됐을 수 있다. **구역 3이 생명 재획득처를 제공**한다.

| 획득처 | 좌표/지대 | 방식 | order 근거 |
|---|---|---|---|
| **생명의 샘물 E** | (12,22), 뿌리 회랑(GH1 뒤·GH2 앞) | idempotent `add_vita(1)`(이미 보유 시 재지급 안 함) | GH1 통과 후·GH2 도달 전 경로에 강제 배치 → 심장 정화 전 확보. L5 재획득처 A/B(발전 제단/마력 성물함) 패턴 복제 |

- **왜 회랑에 두는가**: GH1과 GH2 사이 유일 경로(뿌리 회랑)에 놓아, 최심부로 올라가는 도중 반드시 지난다 → 생명 재획득이 동선에 자연 포함. 신규 재화 시스템 0(생명=이미 L5에서 vita 자릿수 구현). idempotent = 중복 파밍 불가, 보유 보장만.
- **주의**: 생명의 샘물은 **재획득(보상)**이지 게이트가 아니다 → BFS 순서 강제와 무관(§A-6.1 벽 취급 안 함, 인접 reach). 최초 획득처는 여전히 시작의 숲 G4(정사) — 여기는 소진 세이브용 안전망.

## B-3. 무결성 자체 검증

> `game/data/recipes.json`(220종) + `tools/l1x_recipes.py` 대조로 확인. 결정마다 왜 안전한지 논증.

### (1) 페어 중복 없음 (기존 220종 대조 + 신규 내부)
- **원리**: 조합은 `frozenset(inputs)` 무순 2원소 매칭(SAME-쌍은 `('SAME',elem)` 키). 신규 33쌍 전부 검사.
- **I10~I17을 포함한 쌍은 구조적 충돌 불가** — 기존 220종은 I10~I17을 안 쓴다. 위험은 순수 D 쌍(교차 R27~R30의 D19/D21/D22, 2차 R08/R09/R13/R14/R19/R20/R25 등 D+D/D+I)인데, 신규 D222~ 산출은 전부 EX-L1 전용이라 기존 어디에도 없어 안전.
- **자기 충돌**(신규 내부): **초안에서 5건 검출**(푸른 물감 vs 색 모래 반죽 / 색 이슬차 vs 노란 물감 / 물든 모래병 vs 붉은 물감 / 심장 이끼차 vs 맑은 수액 / 첫 실험의 잔재 vs 생명의 씨눈) → 재료 교체(D-중간재 경유·페어 재배치)로 해소, **최종 내부 중복 0**.
- **SAME-쌍 안전**: R15~R17·R21~R23(희귀 꽃²/이슬²/색 모래²/수액²/씨눈²/이끼²)은 기존 SAME-쌍(시작의 숲 등)과 원소값이 달라 충돌 아님.

### (2) 게이트 체인 softlock 불가 (Part A §A-6과 연결)
- 채집 원소 체인은 §A-6.2 표(모든 '게이트 前?' = ✅)가 증명. Part B는 그 원소를 구체 레시피로 전개했을 뿐 뒤-지대 의존 추가 없음. `l1x_recipes.py` softlock 블록 = 전 게이트 `OK`.
- **유니크 I14 특례**: I14는 GH2 최종물(되살아난 심장)에만 쓰이고 GH2는 마지막 게이트 → 앞 게이트(GH1)를 막지 않음(§A-6.2). economy "키 파생물 조건부 Echo"(첫 파생 D233 복제 허용)로 실수 소진 방지 옵션.

### (3) 도달성 (I10~I17 전부 게이트 前 지대에 채집원 존재)
- §A-6.2 대표 좌표가 각 원소를 그 체인 여는 게이트 前(누적) 지대에서 보증. §A-6.1 BFS orphan 0(양 구역, "orphan objects: NONE").

## C-2. 구현 단계 스케치 (EX-L1-1 ~ EX-L1-N)

> L2~L5 마일스톤 분해 패턴 계승. 구역별 병렬 가능(화원·심장 독립).

1. **EX-L1-1 타일/오브젝트** — 화원(파스텔 리컬러 + 무지개 분수/시든 아치/색맞춤 화단 슬롯 + 채집 f/d/z/y) · 심장(심부 리컬러 + 세계수 심장/생명의 샘물/뿌리문/심장 봉인 + 채집 j/e/q). whisper_tileset source 추가(기존 규약 유지). `l1x_map_gen.py`가 layout+legend(+height) 산출.
2. **EX-L1-2 맵 로더** — `l1g_*`·`l1h_*` 6파일 경로 추가. L4/L5 로더 무수정 재사용(고도 균일 0). legend gates 블록 = §A-3 게이트 셀/타입 직렬화.
3. **EX-L1-3 게이트 로직** — 전부 시작의 숲/L2~L5 시그널 재사용(§C-1). GA1/GA2/GA4/GH1/GH2 = node_id·item_id 교체. **GA3 = GatePuzzle 색맞춤 술어**(3슬롯 아이템=빨/노/파 매칭, 순서 무관, 재배치 허용) 신규 술어 1개.
4. **EX-L1-4 데이터 반영** — `items.json`에 I10~I17(`layer:1`, gather, I14 unique) + D222~D254(`layer:1`, 조합/placement) 추가. `recipes.json`에 EX-L1-R01~R33 추가(`l1x_recipes.py` 출력). 아이콘 41종. `l1x_data_fragment.json`이 손입력 없이 제공.
5. **EX-L1-5 NPC + 진상 조각** — `quests.json` N- 라인 2개(정원사 석상·첫 컨스트럭터 잔향, 각 3~5 체인). 진상 조각 2개(정원사 석화·첫 실험 흔적) TruthShard 등록. 화원 조각 = 톤, 심장 조각 = **엔딩 5조각 구조의 L1 조각**(세계수/EG 진상 심화).
6. **EX-L1-6 생명 재획득 + 포탈 + 흐름** — 생명의 샘물 E `add_vita(1)` idempotent(L5 A/B 복제). 시작의 숲↔화원↔심장 포탈 연결점(개방 조건=layer1_purified). QuestManager에 EX-L1 라인 편입, 세이브 플래그(garden_purified·heart_purified), NG+ reset 편입. **엔딩 Balance 4축에 생명 재확보 경로 반영**.

## C-3. 낮/밤·틴트 구현 (시작의 숲 C-3 계승)

- 화원·심장 모두 시작의 숲 CanvasModulate 시간 틴트 재사용. **추가 틴트 커브**: 화원 = 정화 진행도(게이트 수)에 따라 **채도 복귀 커브**(GA1 앞 거의 무채색 → 신전 무지개). 심장 = 심부 하강 **어둠+보라 발광 커브**(row 낮을수록 어둡게, 뿌리 도랑/심장 코어 additive glow는 CanvasLayer 분리로 살아남). "지하 -1 고도"는 이 틴트로만 표현(실제 고도 미사용).
- 발광 오브젝트(무지개 분수·심장 코어·뿌리 도랑) = additive 레이어 분리(시작의 숲 C-3 계승) → 심부/야간에 base가 어두워질 때 발광만 도드라짐.

## C-4. 컷신 1개 — 구역 3 최심부 「다시 뛰는 심장」 (GH2 봉헌 직후)

> cutscenes-v2 포맷 계승(짧은 비트, 무언 주인공, Whisper가 유일한 발화). L1 심부 서사의 정점 + 첫 컨스트럭터 잔향의 작별. 구현은 clear_sequence 훅 재사용.

**시퀀스** (되살아난 심장 D235를 심장 봉인 목에 봉헌 = GH2 클리어)

1. **봉헌** — 되살아난 심장을 봉인 목에 놓으면, 멈춰 있던 세계수 심장(최심부 코어)이 화면 중앙에서 **한 번, 크게 고동친다**(저음 심장박동 1회 — 오프닝 CS-01의 보라 점 박동과 수미상관). 화면이 잠깐 어두워졌다, 심장 박동에 맞춰 보라·초록 발광이 번진다.
2. **되살아남** — 뿌리 도랑을 따라 생명수가 다시 흐르기 시작하고(도랑 발광이 남→북으로 옮겨 붙음), 마른 뿌리들이 아주 느리게 감긴다. 최심부 전체에 온기가 돈다. (박동 2회째 — 이제 규칙적으로.)
3. **첫 컨스트럭터의 작별** — 심장 곁 잔향 NPC(첫 컨스트럭터)가 마지막으로 희미하게 밝아지며 말한다: *"…아, 이렇게 하는 거였구나. 나는… 심장에서 꺼내기만 하고, 돌려주는 걸 잊었어. 너는, 잊지 마."* 잔향이 심장 빛에 스며들 듯 옅어진다(석상/잔영 계열의 '작별' 연출).
4. **진상 접속** — 첫 실험 흔적(랜드마크 3)이 잠깐 밝아지며 **L1 심부 진상 조각**이 확정 획득(아직 안 모았다면). 세계수(EG 진상)의 이야기 = "선배들도 되살리려 했으나, 꺼내기만 하고 돌려주는 법을 몰랐다" — 이는 **L5 '응답'(세계에 대답하기)의 자연 챕터 예고**(창조자가 세계에서 꺼내기만 한 죄 → L5에서 처음 되돌려줌). 스토리라인 §2 진상·§5 엔딩과 정합.
5. **복귀** — 심장 박동이 은은히 이어지는 채로 컷신 종료(`time_running` false→true 페어링, clear_sequence 패턴). 최심부 개방(탐험 자유), 심장 정화 플래그(heart_purified) 세팅. 홈 귀환 시 L1 확장 완료 반영.

- **주제 정합**: 시작의 숲 G4 = "비워낸 자리에 다시 생명을"(심는 것). 심장 GH2 = "심장에서 꺼낸 걸 심장에 돌려주는 것"(돌려주는 것). **꺼냄→돌려줌의 대비**가 L1 안에서 완성되고, 이는 게임 전체 주제(창조자가 세계에서 무엇을 꺼내고 무엇을 돌려주는가 = L5 '응답'의 뒤집힘)의 **첫 씨앗**이다. L1 확장이 단순 볼륨이 아니라 **엔딩 주제의 복선**을 심는다.
- **왜 이 컷신 1개인가**: 화원 GA4 정화는 "색이 돌아오는" 시각적 보상으로 충분(별도 대사 컷신 불필요, 색 번짐 연출만). 심장 최심부는 **진상 서사 + 엔딩 복선**이 걸려 있어 대사 컷신이 필요 — Part C 컷신 1개를 여기 배정.
