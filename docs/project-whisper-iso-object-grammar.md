# Project Whisper — 아이소메트릭 오브젝트 문법 v0.1 (draft)

> 작성: Kana / 2026-07-07 · 상태: **검수 대기** (v1.2.0 아트 정합 게이트 1)
> 근거: 오너 지적 — 바닥은 2:1 아이소(128×64 다이아)인데 오브젝트가 정면뷰라 원근 충돌("종이 팻말").
> 이 문서는 [art-style-guide](project-whisper-art-style-guide.md)의 하위 규약. 팔레트·selout·라이팅 방향은 상위 가이드가 우선.

---

## 1. 왜 필요한가
바닥 타일은 3/4 탑다운 아이소(2:1 다이아몬드). 그런데 기존 오브젝트 다수가 **정면 정투영**(front orthographic)이라 바닥 평면과 각도가 안 맞아 "세워둔 팻말"처럼 뜬다. 특히 기계 계열(gear_assembly / gen_sub / rune_pillar)이 최악, 유기물(나무/바위)은 실루엣이라 상대적으로 무난.

## 2. 핵심 규약 — 3/4 아이소 박스 문법

**오브젝트 = 3면이 보이는 입방체/실린더로 생각한다.** 정면 1면이 아니라:
- **윗면 (top face)** — 마름모. **바닥 다이아 각도와 평행**: 좌우로 1px 갈 때 상하 0.5px (기울기 2:1). 폭:높이 = 2:1.
- **좌측면 (left face)** — 세로 벽, 아래로 내려갈수록 살짝 오른쪽 하강.
- **우측면 (right face)** — 세로 벽, 아래로 내려갈수록 왼쪽 하강.

실린더(솥·기둥)는 윗면=타원(rx:ry = 2:1), 몸통=수직 벽으로 좌/우 톤 분리.

## 3. 접지 (grounding)
- 오브젝트 origin = 캔버스 하단중앙, 바닥 셀 다이아 중심에 안착 (art-guide §1 그대로).
- 풋프린트가 셀 안에 들어오는 **마름모 밑변**을 갖고, 그 밑에 **접지 그림자(타원 블롭)** — 기존 `ao()` 헬퍼 (검정, 소프트, rx:ry≈3.5:1). 이게 "바닥에 놓였다"를 만든다.

## 4. 라이팅 — **상위 가이드 준수: 우상단(북동) 고정**
> ⚠️ 착수 브리프에는 "북서(NW)"라 적혀 있었으나, art-style-guide §2가 **"우상단 소프트 라이트 고정, 모든 타일·오브젝트·캐릭터 동일 방향"**을 STRICT로 못박음. 기존 전체 에셋이 우상단이라 여기서 뒤집으면 오히려 정합이 깨진다. **따라서 광원은 우상단(NE) 고정**으로 확정. (3톤 셰이딩 원칙 자체는 동일.)

3톤 셰이딩:
| 면 | 밝기 | 계산 |
|---|---|---|
| 윗면 (top) | **최명** | base → HI 쪽으로 |
| 우측면 (광원측) | 중간~밝음 | base 근처, 상단 rim 하이라이트 |
| 좌측면 (그늘측) | **최암** | base → SH 쪽으로 |

## 5. 윤곽 (outline)
- 1px, **순수 검정 금지** (art-guide §2 selout). 각 면 색보다 2단계 어두운 동일 계열.
- 내부 모서리선(윗면/측면 경계)은 SH 톤으로 1px, 과하지 않게.

## 6. 팔레트
기존 레이어별 램프 그대로 사용 (변경 금지). L1 nature 22색 / L2 navy+cyan / L3 copper+brass+orange / L4 rune violet+gold / L5 왜곡 시안·마젠타. 새 색 도입 없음 — **형태(뷰)만 교정**.

## 7. 유기형 예외 (statue 등)
유기형은 박스로 못 짜맞추므로 실루엣 유지하되 **접지·광원·그림자 3개만 통일**: 밑에 마름모 풋프린트+타원 그림자, 우상단 광원 3톤, selout 아웃라인. 원근은 "약하게 뒤로 기운 3/4 자세"로 정면감만 뺀다.

## 8. 체크리스트 (자가 검수 항목)
1. 윗면 마름모/타원 기울기가 바닥 다이아(2:1)와 평행한가?
2. 3면(윗/좌/우) 톤이 우상단 광원으로 분리됐는가?
3. 밑에 타원 접지 그림자가 있는가?
4. 아웃라인이 순수 검정이 아닌 selout인가?
5. 팔레트가 해당 레이어 램프 안인가? (신규 색 없음)
