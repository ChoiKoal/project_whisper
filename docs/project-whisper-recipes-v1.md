# Project Whisper — 조합 트리 & 퍼즐 설계 v1.0 (MVP)

> 작성: Kana / 2026-07-05
> 참고 계보: Little Alchemy / Doodle God 류의 원소 조합 문법을 1구역 재료에 이식
> KOAL 지시 반영: "탐험형 + 기본적으로 퍼즐링이 되어야 함" → **조합이 탐험의 열쇠가 되는 구조**

---

## 1. MVP의 목표 (게임의 '왜')

**"세계수에 도달해, 채집으로 비워낸 자리에 어린 세계수를 되심어라."**

- 탐험 중 길이 막힘 → "뭘 조합해야 열리지?" → 재료 찾아 탐험 → 조합 발견 → 길 열림 → 반복
- 채집할수록 세계에 VOID(구멍)가 늘어남 → 마지막에 그 VOID에 생명을 되심는 것이 클리어 → 채집=소비, 창조=회복이라는 테마가 튜토리얼 구역에서 완결됨

## 2. 퍼즐 게이트 4종 (시작의 숲)

| # | 게이트 | 열쇠 | 가르치는 것 |
|---|---|---|---|
| G1 | 시냇가 — 물 타일이 길을 끊음 | **디딤돌**(바위+돌)을 물 타일에 배치 | 아이템을 세계에 '배치'하기 |
| G2 | 마른 덤불이 길을 막음 | **물 아이템**을 덤불에 사용 → 개화하며 통로 열림 | 아이템을 오브젝트에 '사용'하기 |
| G3 | 세계수 구역 입구 | **저녁~새벽 시간대**에만 꽃길이 열림 (기획 문서 그대로) | 시간 시스템 관찰 |
| G4 | 클리어 | 세계수 채집(1회 유니크) → 생명수 → 빛나는 새싹 → **어린 세계수를 VOID 타일에 심기** | 조합 체인 + 테마 완결 |

순환 의존 없음 검증: G1·G2는 기본 채집물만으로 해결 가능 → G3은 시간만 → G4 재료(세계수)는 G3 통과 후 획득.

## 4. 힌트 시스템

- 실패 조합 시 재료 미소모 + 힌트 게이지 +1
- 게이지 5회 누적 → 미발견 레시피 하나의 **재료 한쪽 실루엣** 공개
- 게이트 앞에서는 환경 힌트: G1 시냇가 옆 "바위와 돌이 쌓인 흔적", G2 덤불 "말라 있다… 목말라 보인다" 플레이버 텍스트
