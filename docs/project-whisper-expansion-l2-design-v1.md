# Part A — 「지하 데이터 성소」 40×40 타일 설계

## A-1. 컨셉

터미널 스테이션에서 관제탑을 되살린 방랑자가, 관제탑단 아래로 내려가는 정비 승강로를 따라 다다르는 곳. **정전된 문명의 마지막 백업이 잠든 지하 서버 사원**이다. 관제탑이 기지의 "지금"을 켰다면, 그 아래에는 문명의 "기억" 전부가 — 진보와 전쟁과 폐허의 로그가 — 냉각 젤에 잠긴 채 부식되고 있다. 방랑자는 **끊긴 데이터를 한 조각씩 이어** 최심부의 백업 코어에 도달하고, 문명이 남긴 단 하나의 온전한 기록을 되살린다.

터미널 스테이션이 "전력을 복구하는 곳(기계의 정적)"이었다면, 성소는 그 아래에서 **기억을 복원하는 곳** — 데이터 결정·광섬유·부식 코어가 주역이 되는, L2 안의 지하 **아카이브 챕터**다. 구역 1의 남색+시안 무드를 **더 어둡게(지하)** 가라앉히되, 살아 있는 것은 오직 데이터 결정과 백업 코어의 시안 발광뿐이다.

무드: 구역 1 남색 base 위에 **지하의 심연 어둠 + 데이터 시안 발광**. 심부로 내려갈수록(북으로 올라갈수록 = 최심부) 화면이 어둑해지고 서버 랙의 시안 명멸이 강해지는 CanvasModulate 커브. **엔진 틴트(CanvasModulate) + 리컬러**로 팔레트를 늘리지 않고 표현(아트가이드 §2·§3). "지하 -1 고도"는 이 틴트로만 표현(실제 고도 미사용 — 로더 무수정, L1 확장 심장 구역 방식 계승).

**진입점**: 터미널 스테이션 관제탑단(row 0~3 O 블록) 하부의 **정비 승강로** → 성소 남쪽 스폰 (19,39). 터미널 스테이션 클리어(G4 control_core 급전) 후 개방(승강로에 잔류 전력이 돌기 시작 = 개방 신호). 왕래는 홈/구역 포탈 패턴 재사용(§Part C).

## A-6.2 재료 도달성 — 각 게이트 열쇠 재료 소스가 게이트 前 지대 (softlock 불가)

## A-6.3 에너지 Whisper 재획득처 — 엔딩 Balance 대비 (order-safe)

endgame Balance는 4축(자연/**에너지**/마력/생명)이며, E1/E2 판정에 에너지 보유가 관여한다. 터미널 스테이션 G2(보조 발전기 수리)가 에너지를 처음 주지만(layer2 §보완), 회차·진행에서 **소진**됐을 수 있다. **성소가 에너지 재획득처를 제공**한다.

| 획득처 | 좌표/지대 | 방식 | order 근거 |
|---|---|---|---|
| **잔류 전력 노드 E** | (12,22), 랙 회랑(GB1 뒤·GB2 앞) | idempotent `add_energy(1)`(이미 보유 시 재지급 안 함) | GB1 통과 후·GB2 도달 전 경로에 강제 배치 → 백업 봉헌 전 확보. 구역 1 G2 보상 패턴 + L1 확장 생명의 샘물 idempotent 패턴 복제 |

- **왜 회랑에 두는가**: GB1과 GB2 사이 유일 경로(랙 회랑)에 놓아, 최심부로 올라가는 도중 반드시 지난다 → 에너지 재획득이 동선에 자연 포함. 신규 재화 시스템 0(에너지=이미 WhisperCurrency 자릿수 구현). idempotent = 중복 파밍 불가, 보유 보장만.

## C-4. 컷신 1개 — 최심부 「깨어나는 백업」 (GB4 봉헌 직후)

> cutscenes-v2 포맷 계승(짧은 비트, 무언 주인공, Whisper가 유일한 발화). L2 심부 서사의 정점 + 관리 드론의 작별. 구현은 clear_sequence 훅 재사용(구역 1 G4 관제탑 재가동 컷신과 동형).

**시퀀스** (복원 코어 D263를 백업 봉헌 목 H에 봉헌 = GB4 클리어)

1. **봉헌** — 복원 코어를 봉헌 목에 놓으면, 부식된 채 약하게 명멸하던 백업 코어(최심부 O)가 화면 중앙에서 **한 번, 깊게 점등한다**(저음 부팅 톤 1회 — 오프닝 CS-01의 각성 점멸과 수미상관). 화면이 잠깐 암전됐다, 코어 점등에 맞춰 시안 발광이 서버 랙을 타고 번진다.
2. **되살아남** — 광섬유 트레이를 따라 데이터가 다시 흐르기 시작하고(트레이 발광이 최심부→어귀로 남하하며 옮겨 붙음), 죽어 있던 서버 랙들이 순차 점등한다. 성소 전역에 기록의 빛이 돈다. (부팅 톤 2회째 — 이제 규칙적으로.)
3. **관리 드론의 작별** — 코어 곁 관리 드론 NPC가 마지막으로 희미하게 밝아지며 말한다: *"…접근 권한, 확인… 됐어. 백업이, 깨어났어. 나는 이걸 지키라고만 남겨졌는데… 이제야, 지킨 게 뭔지 알겠어. 너는… 잊지 말고, 가."* 드론이 코어 빛에 스며들 듯 옅어진다(로봇/잔영 계열의 '작별' 연출 — EX-L1 첫 컨스트럭터 작별과 동형).
4. **진상 접속** — 전쟁 기록 단말(랜드마크 3)이 잠깐 밝아지며 **L2 심부 진상 조각**이 확정 획득(아직 안 모았다면). 마지막 백업이 담은 로그 = "문명은 자원을 두고 싸우다 전력을 잃었고, 마지막까지 남긴 것은 무기가 아니라 이 기억이었다" — 이는 endgame 진상 5조각(자연/에너지/마력/생명/응답) 중 **에너지(L2) 조각**을 완성하고, 스토리라인 §2 진상·§5 엔딩과 정합(진보→전쟁→폐허의 기록이 되살아나야 엔딩에서 "무엇을 되돌려줄지" 판단 가능).
5. **복귀** — 서버 랙 명멸이 은은히 이어지는 채로 컷신 종료(clear_sequence 패턴). 성소 개방(탐험 자유), 정화 플래그(sanctum_purified) 세팅. 홈 귀환 시 L2 확장 완료 반영.

- **주제 정합**: 구역 1 G4 = "죽은 문명이 마지막으로 한 번 숨을 쉰다"(전력을 켜는 것). 성소 GB4 = "그 문명이 남긴 기억을 되살리는 것"(기억을 되돌려주는 것). **전력→기억의 대비**가 L2 안에서 완성되고, 이는 게임 전체 주제(창조자가 세계에서 무엇을 꺼내고 무엇을 돌려주는가 = L5 '응답')의 **에너지 챕터 씨앗**이다. EX-L1 심장 컷신(꺼냄→돌려줌)의 SF 대응물.
- **왜 이 컷신 1개인가**: GB1~GB3 돌파는 발판·격벽 개방·데이터 문의 시각적 보상으로 충분(별도 대사 컷신 불필요). GB4 최심부는 **진상 서사(에너지 조각) + 엔딩 복선 + NPC 작별**이 걸려 있어 대사 컷신이 필요 — Part C 컷신 1개를 여기 배정.
