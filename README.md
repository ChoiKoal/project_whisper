# Project Whisper

멸망해가는 세계들을 탐험하며 원소를 모으고, 조합(Fusion)과 속삭임(Whisper)으로 나만의 세계를 창조하는 아이소메트릭 도트 어드벤처.

- 엔진: Godot 4.5 / GDScript
- 플랫폼: macOS / Windows (Steam 목표)
- 문서: `docs/` (기획서, 아트 가이드, 개발 플랜, 조합 트리)
- 게임 프로젝트: `game/` — Godot 에디터에서 열고 F5

## 실행
1. [Godot 4.5](https://godotengine.org/download) 설치
2. 프로젝트 열기 → `game/project.godot`
3. F5 (메인 씬 실행)
